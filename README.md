# MAX30102 Heart Rate Monitor

This project measures heart rate using the MAX30102 sensor and displays BPM on a 16x2 LCD.

## Components
- MAX30102 Sensor
- Arduino Nano
- 16x2 I2C LCD
- Jumper Wires

## Features
- Real-time BPM measurement
- Finger detection
- LCD display output

## Libraries Used
- Wire
- MAX30105
- heartRate
- LiquidCrystal_I2C
